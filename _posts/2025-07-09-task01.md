# How to **create** a **GitHub Pages Site**?

Ab) This post is a beginner's guide to starting with GitHub pages.

Hello, all.

If you're looking to create a personal website as a developer, **GitHub Pages** is the perfect tool. 
GitHub Pages is ~~notorious~~ for its difficult usage, but follow these steps, then can make simple website!

In this post, we'll cover:
  - Creating a **repository**
  - Enabling GitHub pages in **settings**
  - Adding first post as **'_posts/YYYY-MM-DD-title.md'**

---

## 1. Create a Repository

I'm certain that you already have GitHub account but if you not, create an account first > [GitHub](https://github.com/)

Done? Let's get started.
